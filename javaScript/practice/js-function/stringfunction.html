人到中年，唯心独醉

人到中年，走过风雨，经过别离，尝过伤痛，才真正品味到，淡字的真正涵意。平淡的生活很静，淡雅的花很纯。清亮淡淡的天空很高，夜暮月华淡显幽美。淡淡的孤独很醉人，淡淡的微笑很醇。淡然牵念的友情很真，淡淡的情愁很醇。

四季轮回里，时光如梭，扬手是春，落手是秋。在这一扬一落之间，心中突然有种无言的痛。那种难言的的疼是被青春抛弃的无奈，是被岁月洗尽铅华的不舍。在若有若失，若即若离的相似岁月里，不经意间，年轮已悄悄地在脸上涂上了淡淡的岁月印迹，增添了流年的风霜。

一直认为，童年是梦，少年是歌，青年是诗，那么中年就是一部小说。看着镜中的老去的面容，留恋着内心的太多不舍。蓦然回首，人已中年，心中盈满了太多的温柔思恋，暖暖的，在相似的日月之间共度了人世沧桑，看尽繁华红尘。我们总是认为时间会等我们，容许我们从头再来，弥补人生缺憾，岂不知时光一去不再复返。曾经我们无论倾注了多少真心和热情的日子，最终也只是那么闪亮的一瞬，在生命中留下的只是一缕缕的馨香。虽是一般的小事，却堆积成一个个精彩的故事，激活感动着真真实实我们的生命与心灵。这样的日子，却总在不经意间悄悄远行。

当我们已认不出那些似曾相识的岁月，就好像难以说出那一阵微风曾经轻吻过我们的脸庞。有些事是时间和空间都无法修复的，有些风景，只适合远看。有些人，只适合呆在年少轻往的岁月里。有些事，只能在记忆中，储存在回忆里。有时，虚幻的也许更美。人生就是在希望与失望之中，在得与失里，纠结和伤痛中，完成了成熟的蜕变。

人到中年，一个“悟”字，简述了人生的千言万语。明白了，高处不胜寒，无力挽狂澜的世事沧桑与无奈。于是，在不知不觉中，我们终于学会，笑看世事繁华。淡定人生心态，不求不可取，不求不可得，随遇而安，诸事顺其自然，学会感恩，无憾我心的做人，做事。

当用心触摸真实生活的平淡之时，你就会真实的感觉到只有脚踏实地平平淡淡才是真。学会轻轻细细地触摸生之意义，在闲闲的时间里，静静的写下几段文字给自己或朋友。生活的阅历与磨练，终会打磨出一个人成熟的姿态。在书香里，在茶的氤氲里，风定嗅花香，淡淡的享受，人生成熟的一种优雅之美。

生命像一条河，左岸是我们无法忘却的回忆，右岸是我们值得紧握的璀璨年华，中间飞速流淌的是我们年年岁岁淡淡经历的欢悦与感伤。我们用淡，把平仄婉约的人生写成篇章。拾掇岁月给予的所有沧桑，淡中求永恒，静里思情义淡淡绵长。

人到中年，才知道淡中的意味。只是那么淡淡的一点过去，现在，与将来，在淡里，将人生的画卷轻轻地描绘。绘出天，绘出地，绘出山，绘出水，绘出落下的泪，绘出展开的笑。都用那淡淡的画笔，随着那淡淡的颜色，绘出人生的佳境，绘出人生的无憾无悔。人到中年，唯心独醉，在淡中，静静地守望着，生活与岁月的点滴幸福。
